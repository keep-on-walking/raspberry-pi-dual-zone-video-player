## How to Upload via GitHub Web Interface

### For Each File:

1. Go to: https://github.com/keep-on-walking/raspberry-pi-dual-zone-video-player

2. **Navigate to the file** (e.g., click `install.sh`)

3. **Click the pencil icon** (✏️ Edit)

4. **Delete all existing content**

5. **Open the new file** from this folder on your computer

6. **Copy and paste** the entire new content

7. **Scroll down** and click **"Commit changes"**

8. **Repeat** for each file

## After Upload

Users can now:

```bash
# Fresh install (one command)
sudo apt update && sudo apt install -y git && git clone https://github.com/keep-on-walking/raspberry-pi-dual-zone-video-player.git && cd raspberry-pi-dual-zone-video-player && sudo bash install.sh
```

Then reboot:
```bash
sudo reboot
```

And it will:
- ✅ Hide desktop automatically
- ✅ Videos play from web interface
- ✅ Resize handles work properly
- ✅ Everything works after reboot
